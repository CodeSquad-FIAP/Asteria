# ═══════════════════════════════════════════════════════
# INSTRUÇÕES DETALHADAS PARA ARQUIVO TXT
# ═══════════════════════════════════════════════════════

# 1. FORMATOS ACEITOS:
#    ✅ Produto - Quantidade - Valor - Data
#    ✅ Produto: Quantidade x Valor
#    ✅ Formato livre com números e valores
#
# 2. REGRAS IMPORTANTES:
#    ✅ Linhas começando com # são comentários (ignoradas)
#    ✅ Linhas vazias são ignoradas
#    ✅ Use vírgula ou ponto para decimais (aceita ambos)
#    ✅ Se não informar data, usa data atual automaticamente
#
# 3. O QUE O SISTEMA RECONHECE:
#    - Produtos: texto que sobra após extrair números
#    - Quantidades: números inteiros
#    - Valores: números com decimais, com ou sem R$
#    - Datas: formato AAAA-MM-DD (opcional)
#
# 4. EXEMPLOS QUE FUNCIONAM:
#    Mouse Logitech - 5 - 85.50
#    Teclado: 3x 320,00
#    10 Carregadores por R$ 45.90
#    Vendeu 2 Notebooks a 2500 reais
#
# 5. DICAS PARA MELHOR RECONHECIMENTO:
#    - Seja claro com números e valores
#    - Use palavras como "x", "por", "cada", "unidades"
#    - Evite muitos números no nome do produto
#    - Teste com poucas linhas primeiro
#
# 6. VANTAGENS DO FORMATO TXT:
#    - Mais flexível que CSV
#    - Aceita linguagem natural
#    - Fácil de editar em qualquer editor
#    - Ideal para dados não estruturados
#
